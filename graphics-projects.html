Graphics Programming

These are the programs I have worked on relating to graphics programming. There are a two types of programs I have here. The first type covers different rendering techniques that I've learned about. Some of these rendering techniques are good for more realistic images, some are better suited for stylization, some are used for real-time rendering, and some were just fun to make. The other type of program I worked on was post processing. I used fragment shaders to test out a bunch of different effects to give images more style.
